## Deployment Steps

### Step 1: Prepare Backend
```bash
cd get-together
npm install
```
- [ ] Dependencies installed successfully
- [ ] No error messages

### Step 2: Deploy to AWS
```bash
npx amplify deploy
```
- [ ] Deployment shows success message
- [ ] CLI prompts for:
  - App name: `get-together`
  - Environment: `dev` (or your preference)
  - AWS profile: `default` (or your configured profile)
- [ ] Backend creates:
  - [ ] Cognito user pool
  - [ ] AppSync GraphQL API
  - [ ] DynamoDB tables
  - [ ] `amplify_outputs.json`

### Step 3: Copy Backend Outputs
```bash
cd ..
cp get-together/amplify_outputs.json amplify_outputs.json
```
- [ ] File copied successfully
- [ ] `amplify_outputs.json` exists in frontend root

### Step 4: Configure Environment
```bash
cp .env.local.example .env.local
```
- [ ] Edit `.env.local` with values from `amplify_outputs.json`:
  - [ ] `NEXT_PUBLIC_AWS_REGION` = from config
  - [ ] `NEXT_PUBLIC_USER_POOL_ID` = Cognito pool ID
  - [ ] `NEXT_PUBLIC_USER_POOL_WEB_CLIENT_ID` = Cognito app client ID
  - [ ] `NEXT_PUBLIC_COGNITO_DOMAIN` = Cognito domain
  - [ ] `NEXT_PUBLIC_APPSYNC_GRAPHQL_ENDPOINT` = AppSync endpoint

### Step 5: Install Frontend Dependencies
```bash
npm install
```
- [ ] All dependencies installed
- [ ] No error messages
- [ ] `node_modules/` created

### Step 6: Run Frontend
```bash
npm run dev
```
- [ ] Dev server starts on `http://localhost:3000`
- [ ] No build errors
- [ ] Console shows "✅ Amplify configured successfully" or warning (if outputs not yet loaded)

## Troubleshooting

### Problem: "amplify_outputs.json not found"
**Solution**:
1. Did you run `amplify deploy`? If not, do that first.
2. Is the file in the correct location? Should be in `get-together-web/` root
3. Try: `cp get-together/amplify_outputs.json amplify_outputs.json`

### Problem: "User pool not configured"
**Solution**:
1. Check `.env.local` exists and has `NEXT_PUBLIC_USER_POOL_ID`
2. Verify value matches output from `amplify_outputs.json`
3. Restart dev server: `npm run dev`

### Problem: Login form doesn't appear
**Solution**:
1. Check `app/layout.tsx` has Authenticator component
2. Verify `amplify-provider.tsx` exists
3. Check browser console for errors
4. Try clearing cache: Ctrl+Shift+Delete (Chrome)

### Problem: "Cannot find module '@aws-amplify/...'"
**Solution**:
1. Run `npm install` in frontend root
2. Verify `package.json` has correct Amplify packages
3. Delete `node_modules/` and `.next/` and reinstall
